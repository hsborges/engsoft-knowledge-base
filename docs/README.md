> **Disclaimer:** este acervo e um compilado de documentos relevantes do curso para uso pessoal, sem relacao institucional oficial com a UFMS. Esta organizacao e inspirada no conceito de [LLM Knowledge Bases](https://x.com/karpathy/status/2039805659525644595).

Este diretorio e a porta de entrada da base documental do curso de Engenharia de Software. Aqui voce encontra os caminhos principais para navegar por documentos do curso, FAQs operacionais e legislacao de apoio academico.

A leitura recomendada e comecar pelo mapa tematico e seguir para a pasta mais proxima da sua necessidade. Cada subpasta tem um README proprio com contexto, organizacao e links para os arquivos de referencia.

## Como usar este indice

- Primeiro filtro: escolha a pasta por tema no mapa tematico.
- Segundo filtro: abra o README da pasta para navegar no inventario local.
- Para busca textual, use termos curtos e sinonimos (ex.: prazo maximo, regime especial, ppc).
